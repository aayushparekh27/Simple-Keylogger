# ⌨️ Keystroke Logger (Python Project – Educational)

A basic **Keystroke Logger** built using **Python** and the `pynput` library. This project captures keyboard input and stores it in a file, demonstrating how low-level input monitoring works.

---

## ⚠️ Important Disclaimer

This project is created **strictly for educational and ethical learning purposes only**.
Do **NOT** use this tool to monitor others without their knowledge or consent. Unauthorized use may be illegal and unethical.

---

## 🔥 Project Overview

This project demonstrates how keyboard events can be captured using Python. It logs keystrokes into a file, helping learners understand concepts related to **input handling and system-level monitoring**.

---

## ✨ Features

✔️ Captures keystrokes in real-time
✔️ Logs both character keys and special keys
✔️ Stores output in a text file
✔️ Runs continuously until stopped
✔️ Lightweight and simple implementation

---

## 🛠️ Tech Stack

* **Language:** Python 🐍
* **Library:** pynput

---

## ⚙️ How It Works

* Program starts listening to keyboard input
* Every key pressed is captured
* Characters are written directly to a file
* Special keys (Enter, Space, etc.) are logged with labels
* Logging continues until the program is stopped
